Skills Saya

HTML

HTML (HyperText Markup Language) adalah bahasa dasar untuk membuat struktur dan kerangka halaman web, seperti judul, paragraf, gambar, dan tautan.

CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan desain halaman web, seperti warna, ukuran, font, dan layout agar website terlihat lebih menarik dan rapi.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaksi dan fungsi dinamis pada halaman web, seperti tombol yang bisa diklik, form yang bisa diproses, hingga animasi.
